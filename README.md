# E-Commerce Microservices Backend

A scalable E-Commerce backend application built using Spring Boot Microservices Architecture. The system is designed using industry-standard microservice patterns including Service Discovery, API Gateway, and Centralized Configuration.

## 🚀 Technologies Used

* Java
* Spring Boot
* Spring Cloud
* Eureka Discovery Server
* API Gateway
* Config Server
* MySQL
* Maven
* REST APIs

## 🏗️ Architecture

The application is divided into independent microservices:

* Eureka Discovery Server
* API Gateway
* Config Server
* User Service
* Product Service
* Order Service

Each service can be developed, deployed, and scaled independently.

## ✨ Key Features

* Microservices Architecture
* Service Discovery & Registration
* API Gateway Routing
* Centralized Configuration Management
* User Management
* Product Management
* Order Processing
* RESTful API Development
* Database Integration with MySQL
* Exception Handling
* Layered Architecture

## 📂 Project Structure

* Discovery Server
* API Gateway
* Config Server
* User Service
* Product Service
* Order Service

## 🛠️ Setup

1. Start Config Server
2. Start Eureka Discovery Server
3. Start API Gateway
4. Start Individual Microservices
5. Configure MySQL Database
6. Access APIs through API Gateway

## 🎯 Purpose

This project demonstrates real-world microservices development using Spring Boot and Spring Cloud by implementing scalable and maintainable backend services for an E-Commerce platform.
